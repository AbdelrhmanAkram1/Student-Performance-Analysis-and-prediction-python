## 📌 Project Overview
This project is a comprehensive **Data Mining** system designed to analyze and predict student academic performance. By leveraging behavioral, socioeconomic, and academic data, the system identifies key factors that influence success and provides automated, actionable recommendations for educational institutions.

## 🚀 Key Features
* **Exploratory Data Analysis (EDA):** Deep dive into 6,607 student records to find correlations.
* **Advanced Clustering:** Identifying hidden patterns and student segments.
* **Fuzzy Logic Integration:** Producing interpretable and human-like performance scores and recommendations.
* **Optimization:** Utilizing **Genetic Algorithms** to refine and optimize system parameters.
* **Early Warning System:** Helping educators identify at-risk students based on behavioral data.

## 📊 Dataset
The project uses the `StudentPerformanceFactors.csv` dataset, which includes 20 features such as:
* **Academic:** Previous Scores, Study Hours, Tutoring Sessions.
* **Behavioral:** Attendance, Motivation Level, Physical Activity.
* **Socioeconomic:** Parental Education, Internet Access, Family Income.
